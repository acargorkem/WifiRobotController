# WifiRobotController

This app is specially developed for wifi robot control. With this application, data can be exchanged with http methods, mjpeg video streams can be watched.

![screenshot of app](https://acargorkem.netlify.app/img/img2.png)

### Dependencies

[Google volley](https://github.com/google/volley)

[controlwear virtual-joystick-android](virtual-joystick-android)

[perthcpe23 android-mjpeg-view](https://github.com/perthcpe23/android-mjpeg-view)
